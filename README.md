# nextjs-pdf-viewer
<br>
<br>

## Getting Started

Clone the repo :
```shell
git clone https://github.com/htchoi1006/nextjs-pdf-viewer
cd nextjs-pdf-viewer/pdf-viewer
```

<br>

install dependencies : <br>
yarn :
```shell
yarn
```

npm :
```shell
npm run install
```
<br>

## Running the Example

yarn :
```shell
yarn dev
```

npm :
```shell
npm run dev
```
